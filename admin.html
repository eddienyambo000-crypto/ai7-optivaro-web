GitHub Connection

Connect your GitHub account once. This lets you save changes directly to the live website.

One-time setup — takes 2 minutes

  1. Go to github.com → Settings → Developer settings → Personal access tokens → Tokens (classic)
  2. Click "Generate new token (classic)"
  3. Give it a name like ai7-admin, set expiration to No expiration
  4. Check the repo box (full repository access)
  5. Click "Generate token" and copy it
  6. Paste it below — it's saved on this device only
GitHub connected — changes go live in ~30 seconds after saving
